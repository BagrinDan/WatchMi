# WatchMi 0.1.0 🎬

**WatchWithMi** is a full-stack movie-tracking and companion application designed to help users manage their watchlists, discover new content, and leverage AI-driven recommendations. The project is built using a modern microservices-inspired architecture and is fully containerized for easy deployment.

---

## 🚀 Tech Stack
- **Authentication with Spring Boot**
- **AI model that can local answer**
